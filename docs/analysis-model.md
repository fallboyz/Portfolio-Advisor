# 분석 모델: 검증된 밸류에이션 지표 + Z-Score 합성

## 설계 원칙

개별 자산의 저평가/고평가 판단에는 학술적으로 검증된 밸류에이션 지표를 사용하고,
최종 실물 자산 vs ETF 비율 결정만 자체 로직으로 한다.

## Z-Score 개요

Z-Score = "현재 값이 평균에서 표준편차 몇 개만큼 떨어져 있는지"
- +2: 평균보다 표준편차 2개 위 (과열)
- -2: 평균보다 표준편차 2개 아래 (저평가)
- 0: 평균 수준

## 사용 지표

### 검증된 밸류에이션 지표 (주요)

| 지표 | 대상 | 데이터 소스 | 의미 |
|------|------|-----------|------|
| 10년 실질금리 | 금/은 | FRED (REAINTRATREARAT10Y) | 실질금리 높을수록 금에 불리 (역상관). Z-Score 부호 반전 적용 |
| M2/금 비율 | 금/은 | FRED (M2SL) / 금 가격 | 통화량 대비 금 가격. 높을수록 금이 저평가 |
| CAPE (Shiller P/E) | S&P500 | Shiller Excel | 경기조정 P/E. 높을수록 주식 과평가. 전체 역사(1871~) 대비 |
| Buffett Indicator | S&P500 | FRED (GDP) / S&P500 | 시총/GDP. 높을수록 주식 과평가 |
| Yield Curve | S&P500 | FRED (DGS10 - DGS3MO) | 장단기 금리차. 음수(역전) = 경기 둔화 신호 |
| GSR (금/은 비율) | 은 | 금/은 가격 | 비율 높으면 은 저평가, 낮으면 은 고평가 |

### 보조 지표

| 지표 | 대상 | 의미 |
|------|------|------|
| 가격 위치 | 금/은/나스닥 | 10년 이동평균 대비 현재 가격 위치 |
| 수익률 Z-Score (10년) | 전체 | 10년 수익률 역사 분포 대비 현재 위치 |
| 수익률 Z-Score (5년) | 나스닥 | 5년 수익률 역사 분포 대비 (50년 데이터 부족) |

## 복합 점수 산출

각 자산별로 검증된 지표가 주요 입력(60~70%), 보조 지표가 나머지를 차지.

**금 복합 점수 (config.toml `[weights_gold]`):**
```
S_gold = 0.30*Z_real_rate + 0.30*Z_m2_gold + 0.20*Z_price_position + 0.20*Z_return_10y
```
- 실질금리 Z-Score는 부호 반전 (높은 금리 = 금에 불리)

**은 복합 점수 (config.toml `[weights_silver]`):**
```
S_silver = 0.25*Z_real_rate + 0.25*Z_m2_gold + 0.15*Z_price_position + 0.15*Z_return_10y + 0.20*Z_gsr
```

**S&P500 복합 점수 (config.toml `[weights_sp500]`):**
```
S_sp500 = 0.35*Z_cape + 0.25*Z_buffett + 0.15*Z_yield_curve + 0.25*Z_return_10y
```

**나스닥100 복합 점수 (config.toml `[weights_ndx]`):**
```
S_ndx = 0.40*Z_return_10y + 0.40*Z_return_5y + 0.20*Z_price_position
```

> 일부 Z-Score가 데이터 부족으로 계산 불가한 경우, 사용 가능한 항목만으로 자동 renormalization.

## 2단계 비율 조정 신호

### Level 1: 실물 자산 vs ETF

**그룹 점수:**
```
S_precious = (S_gold + S_silver) / 2
S_etf = S_sp500 * 0.7 + S_ndx * 0.3
R_group = S_precious - S_etf
```

| R_group 구간 | 신호 | 실물 자산 비중 |
|-------------|------|-------------|
| R < -2.0 | 실물 대폭 확대 | 75% |
| -2.0 ~ -1.0 | 실물 확대 | 60% |
| -1.0 ~ +1.0 | 균형 | 50% |
| +1.0 ~ +2.0 | ETF 확대 | 35% |
| R > +2.0 | ETF 대폭 확대 | 20% |

### Level 2a: 금 vs 은

Gold/Silver Ratio Z-Score 기반:

| GSR Z-Score | 금:은 비율 |
|------------|-----------|
| Z > 1.0 | 30:70 (은 크게 저평가) |
| 0 ~ 1.0 | 40:60 |
| -1.0 ~ 0 | 50:50 |
| Z < -1.0 | 60:40 (금이 더 매력적) |

### Level 2b: S&P 500 vs 나스닥 100

상대 복합 점수 차이 기반:

| S_sp500 - S_ndx | S&P:나스닥 비율 |
|-----------------|---------------|
| 차이 > 0.5 | 40:60 (나스닥 저평가) |
| -0.5 ~ 0.5 | 50:50 |
| 차이 < -0.5 | 60:40 (S&P 저평가) |

## 폭락/폭등 보정 (Drawdown/Rally Overlay)

| 조건 | 보정 |
|------|------|
| 은 고점 대비 -40% 이하 | 실물 점수에 -0.5 (더 매력적) |
| ETF 고점 대비 -30% 이하 | ETF 점수에 -0.5 (더 매력적) |
| 은 6개월 +100% 이상 | 실물 점수에 +0.5 (과열 경고) |

## Z-Score 부호 방향 해석

- 점수가 높은(양수) 자산 = 고평가 가능성
- 점수가 낮은(음수) 자산 = 저평가 가능성
- R이 음수: 실물이 저평가, ETF가 고평가 -> 실물 비중 올림
- R이 양수: 실물이 고평가, ETF가 저평가 -> ETF 비중 올림
- 실질금리 Z-Score는 부호 반전 적용 (높은 금리 = 금 불리이므로)
